# Rust Shell
Simple shell made in Rust from this [tutorial](https://www.joshmcguigan.com/blog/build-your-own-shell-rust/).

-----------------

### running
```shell
cd rust-shell/
cargo run
> # ready for receive commands
```

## Shell Built-ins
* `exit`
  * Exits the shell.
* `cd`
  * change current directory. 
  * example: 
   ```shell
   > cd rust-shell/src
   ```
